## What Is SomCharge?
SomCharge is the first large-scale **shared powerbank network in Somalia**, built with IoT-powered charging stations that allow customers to borrow a powerbank at one station and return it at another.  
The service is designed for malls, cafes, universities, offices, public spaces, and high-traffic areas.
